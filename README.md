# Number Format Converter using Verilog

## Overview
A Verilog HDL project that converts numbers between Binary, Hexadecimal, and BCD representations.

## Features
- Binary to BCD conversion using the Double Dabble algorithm.
- BCD to Binary conversion.
- Hexadecimal input support.
- Verification through simulation and testbenches.

## Tools Used
- Verilog HDL
- Active-HDL / ModelSim

## Project Structure
- src/ : Verilog source files
- testbench/ : Testbench files
- screenshots/ : Simulation waveforms and results
